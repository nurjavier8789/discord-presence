Caranya

1. bikin dulu di discord dev portal
2. bikin folder baru
3. run cmd `cd` kearah folder yang dituju
4. trus ketik `npm init -y`
5. trus ketik `npm i discord-rpc`
6. bikin file `.js` baru (terserah mau nama apa)
7. bukak file `.js`nya
8. isinya sama seperti `index.js`
9. trus coba deh :)
